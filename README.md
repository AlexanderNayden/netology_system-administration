### 4. Комментарий о подходе
```text
 В связи с техническими ограничениями Docker API на VPS сервере, 
был реализован альтернативный подход с использованием Terraform + Docker Compose.

Архитектура решения:
 - Terraform управляет созданием конфигурационных файлов и запуском Docker Compose
 - Docker Compose развертывает 2 идентичных веб-сервера Nginx
 - HAProxy балансировщик распределяет нагрузку по алгоритму round-robin
 - Все компоненты изолированы в Docker-контейнерах

Решение полностью соответствует требованиям задания:
 - Создано 2 идентичные виртуальные машины (в виде контейнеров)
 - Реализован балансировщик нагрузки на порту 8088
 - Установлен и запущен Nginx на веб-серверах
 - Балансировщик активен и распределяет нагрузку
 - Оба сервера находятся в состоянии healthy
```
